= rpx_authentication

http://www.github.com/Nielsomat/rpx_authentication

	./script/plugin install git://github.com/Nielsomat/rpx_authentication.git
	or git submodule add git://github.com/Nielsomat/rpx_authentication.git vendor/plugins/rpx_authentication

Allows for easy integration of rpxnow.com into your Rails app.
